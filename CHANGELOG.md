## [1.0.0] - December 17, 2025

### Initial Release - Complete Feature Set

#### Added
- ‚úÖ **Core Dividend Tracking** - Track companies declaring dividends in next 10 days
- ‚úÖ **Eligibility Checker** - Determine if buying today qualifies for dividends
- ‚úÖ **Distribution Timeline** - Show Ex-date, Record date, Distribution date
- ‚úÖ **50 Indian Companies** - NSE-listed companies across multiple sectors
- ‚úÖ **Sector Filtering** - Filter by IT, Banks, Energy, Finance, FMCG, Pharma, Auto, Telecom, Infrastructure
- ‚úÖ **Advanced Search** - Search by company name or symbol
- ‚úÖ **Sortable Table** - Click column headers to sort
- ‚úÖ **Company Info Modal** - Detailed company information with financials
- ‚úÖ **ROI Calculator** - 1-year and 5-year return projections
- ‚úÖ **Shareable Cards** - Generate 1080x1920px image for social media
- ‚úÖ **Download Cards** - Download dividend cards as PNG files
- ‚úÖ **Clipboard Share** - Copy cards to clipboard for sharing
- ‚úÖ **Visitor Counter** - Track page visits using localStorage
- ‚úÖ **Responsive Design** - Works on desktop, tablet, mobile
- ‚úÖ **Windows Launcher** - RUN_PROJECT.bat for easy launching
- ‚úÖ **Complete Documentation** - README, Developer guide, User guide, FAQ

#### Technical Stack
- Backend: Python 3.x with Flask 2.3.3
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Image Generation: Pillow (PIL) 10.0.0
- Data Processing: Pandas 2.0.3
- Server: Werkzeug 2.3.7
